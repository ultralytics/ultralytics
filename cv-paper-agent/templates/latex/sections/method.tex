\section{Method}
SC-ELAN comprises three core components: (1) ContextAwareRepConv blocks with multi-branch training and single-branch inference via re-parameterization, (2) LSKA (Large Kernel Spatial Attention) with TSCG (Two-Stage Context Gating) for selective long-range context modeling, and (3) DetectCAI head with class-aware instance reweighting for long-tail mitigation. The ELAN backbone preserves gradient highways through dense layer aggregation.
