# TensorFlow Edge TPU Export (DEPRECATED)

!!! warning "Deprecation Notice"

    **TensorFlow Edge TPU export is deprecated and no longer supported.**

    The EdgeTPU compiler is incompatible with TFLite models generated by ai-edge-torch (our new TFLite export backend). For edge deployment, use **TFLite format** with INT8 quantization instead:

    ```bash
    # Export to TFLite with INT8 quantization for edge deployment
    yolo export model=yolo11n.pt format=tflite int8=True data=coco8.yaml
    ```

## Why was Edge TPU export deprecated?

Edge TPU export was deprecated because:

1. **Compiler incompatibility**: The EdgeTPU compiler doesn't support many modern TFLite ops generated by ai-edge-torch
2. **Limited op coverage**: Even when compilation succeeds, many operations may not be delegated to the EdgeTPU
3. **Better alternatives available**: TFLite with INT8 quantization provides excellent edge performance
4. **Simpler workflow**: Direct PyTorch to TFLite conversion is more reliable than the multi-step EdgeTPU pipeline

## Migration to TFLite with INT8

### Step 1: Export to TFLite with INT8 Quantization

```bash
yolo export model=yolo11n.pt format=tflite int8=True data=coco8.yaml
```

Or in Python:

```python
from ultralytics import YOLO

model = YOLO("yolo11n.pt")
model.export(format="tflite", int8=True, data="coco8.yaml")
```

### Step 2: Use TFLite for Inference

```python
from ultralytics import YOLO

# Load TFLite model
model = YOLO("yolo11n.tflite")

# Run inference
results = model("https://ultralytics.com/images/bus.jpg")
```

## Coral Edge TPU Hardware

If you have Coral Edge TPU hardware and need to use it:

1. **Use older Ultralytics versions**: Versions before the ai-edge-torch migration may still support EdgeTPU export
2. **Manual TFLite conversion**: Create TFLite models using other tools that produce EdgeTPU-compatible ops
3. **Consider alternatives**: The [Coral documentation](https://coral.ai/docs/) provides guidance on compatible model formats
